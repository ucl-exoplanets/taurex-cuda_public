TauREx-CUDA - GPU accelerated atmospheric retrievals
=======================================================


TauREx-CUDA is an open-source BSDv3 plugin for TauREx3.1_, 
providing replacement forward models that exploit nVidia GPU hardware
to accelerate atmospheric retrievals.

The CUDA backend is provided by the PyCUDA_ python library
